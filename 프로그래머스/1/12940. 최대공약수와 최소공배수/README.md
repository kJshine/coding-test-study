```JavaScript
function solution(n, m) {
    const gcd = getGCD(n, m);
    return [gcd, (n * m) / gcd];
}

function getGCD(a, b) {
    return b === 0 ? a : getGCD(b, a % b);
}
```

- 최대공약수를 구하는 방법: getGCD
- 최소 공배수를 구하는 방법: (n \* m) / gcd
- 사실 유클리드 호제법이 뭔지 잘 모르겠고 이해도 안간다. 하지만 최소공배수와 최대공약수가 필요한
  알고리즘 문제가 많으므로 자주 보고 외울 필요가 있어 보인다.
